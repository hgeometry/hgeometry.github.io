HGeometry

Computational Geometry in Haskell

GitHub Discord API Reference
HGeometry is a library for computing with geometric objects in Haskell. It defines basic geometric types and primitives, and implements many algorithms, including:
\( O(n \log n) \) Convex Hull.
\( O(n) \) Smallest enclosing disk.
\( O(n \log n) \) Douglas Peucker polyline line simplification.
\( O(n \log n) \) Delaunay triangulation.
\( O(n \log n) \) Euclidean Minimum Spanning Tree.
\( O(1/\varepsilon^dn\log n) \) Well-Separated pair decomposition.
\( O(\log n) \) Extremal points and tangents on/to a convex polygon.
\( O(n+m) \) Minkowski sum.
\( O(n\log n) \) Closest pair among a set of points.
\( O(nm) \) Fr√©chet distance.
\( O(n) \) Single-source shortest path tree for simple polygons.
